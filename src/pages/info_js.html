Materiały z JavaScripit'a

Czym jest JavaScripit?

Skryptow język oprogamowania stosowany najczęściej na stronach internetowych.
Standard języka JS ustandaryzowało stowarzyszenie ECMA
JavaScript implementuje rzeczy ze specyfikacji Ecma Scripit
ECMAScripit 6+ - na tym będziemy opierać nauke JS

Typy proste w JS (primitivies)

String - tekst, otaczamy ' lub '' (zalecam ')
Numer - liczba
Bigint - duża liczba, większa niż 2 od potęgi 53
boolean - ture lub false
null - wartośc celowo niezdefiniowana
*symbol - tworzy "unikalne opakowanie"
